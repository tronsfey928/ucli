# Examples

This directory contains complete CLI projects generated by `openapi2cli`.

## github-cli

A CLI targeting a curated subset of the [GitHub REST API](https://docs.github.com/en/rest), used as the integration test fixture for openapi2cli.

```bash
cd github-cli
npm install && npm run build

# Fetch a repository
node dist/index.js repos get-repo --owner octocat --repo Hello-World

# List open issues
node dist/index.js repos list-repo-issues --owner octocat --repo Hello-World --state open

# Get a user profile
node dist/index.js users get-user --username octocat
```

Set `API_TOKEN` for authenticated requests:

```bash
export API_TOKEN=ghp_your_personal_access_token
```
